## Run Locally

To run CarZone locally, run this command on your git bash:

Linux and macOS:
```bash
sudo git clone https://github.com/MaciejMi/CarZone.git
```

Windows:
```bash
git clone https://github.com/MaciejMi/CarZone.git
```
